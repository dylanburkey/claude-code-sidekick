## User Experience Principles

1. **Clarity over cleverness** - Users should immediately understand what they
   can do
2. **Progressive disclosure** - Show basic options first, reveal advanced
   features as needed
3. **Fail gracefully** - Always provide helpful error messages and recovery
   paths
4. **Respect user time** - Minimize clicks, loads, and cognitive overhead

## Feature Priorities

When making trade-off decisions, prioritize in this order:

1. **Accessibility** - Must work for all users
2. **Performance** - Must be fast and responsive
3. **Functionality** - Must work correctly
4. **Aesthetics** - Should look polished
